# OpenCode 스킬 사용 가이드

## 스킬이란?

opencode의 Agent Skills은 AI 에이전트가 프로젝트의 규칙과 패턴을 기억하도록 만드는 설정 파일입니다.
스킬을 불러오면 에이전트는 해당 스킬에 정의된 워크플로우를 따라 작업합니다.

---

## 등록된 스킬 목록

### 1. `react-component` — React 컴포넌트 생성

JSX + Tailwind 패턴에 맞춰 새 컴포넌트를 자동으로 생성합니다.

```
skill({ name: "react-component" })
```

**사용 예:**
- "설정 화면 컴포넌트 만들어줘"
- "OptionList에 호버 효과 추가해줘"

### 2. `game-data` — 게임 데이터/스토리 수정

스테이지, 선택지, 트레잇 시스템의 데이터 구조를 안전하게 수정합니다.

```
skill({ name: "game-data" })
```

**사용 예:**
- "철학 엔딩으로 가는 스테이지 3개 추가해줘"
- "boot 스테이지 선택지에 조건 추가해줘"

### 3. `terminal-ui` — 터미널/사이버펑크 스타일링

프로젝트의 색상, 폰트, 애니메이션 규칙을 적용합니다.

```
skill({ name: "terminal-ui" })
```

**사용 예:**
- "이 컴포넌트에 글리치 효과 넣어줘"
- "전체 UI 배경색 약간 밝게 조정해줘"

### 4. `full-feature` — 메타 스킬 (완전 기능 구현)

위 3개 스킬을 순차적으로 불러와 **기획 → 데이터 → 컴포넌트 → 스타일 → 검증**까지 한 번에 수행합니다.

```
skill({ name: "full-feature" })
```

**사용 예:**
- "인벤토리 시스템 추가해줘" — 데이터, 컴포넌트, UI까지 전부 한 번에
- "세이브/로드 기능 만들어줘"
- "NPC 대화 시스템 추가해줘"

## 빠른 사용법

### 한 번에 작업하기 (추천)

복잡한 기능은 `full-feature` 메타 스킬 하나로 충분합니다:

> **"full-feature 스킬을 써서 도감 시스템을 만들어줘. 몬스터 10종 등록하고 UI는 팝업 형태로."**

### 단일 스킬만 쓰기

특정 영역만 작업할 때:

> **"game-data 스킬을 써서 카오스 루트 스토리 5스테이지 분기 추가해줘"**

## 새 스킬 추가하는 법

`.opencode/skills/<이름>/SKILL.md` 파일을 만들면 됩니다.

```markdown
---
name: my-skill
description: 이 스킬이 하는 일 (1024자 이내)
---

## 지시사항
- 규칙 1
- 규칙 2
```

**이름 규칙:**
- 소문자, 숫자, 하이픈(-)만 사용
- 1~64자
- `--` 연속 하이픈 불가
- 폴더명과 `name` 필드가 일치해야 함

---

## 스킬 저장 위치

| 위치 | 경로 |
|---|---|
| 프로젝트 전용 | `.opencode/skills/<이름>/SKILL.md` |
| 글로벌 (모든 프로젝트) | `~/.config/opencode/skills/<이름>/SKILL.md` |

현재 이 프로젝트는 `.opencode/skills/` 아래에 4개 스킬이 등록되어 있습니다.
